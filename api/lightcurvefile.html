LightCurveFile classes¶

Defines LightCurveFile classes, i.e. files that contain LightCurves.

Classes¶

KeplerLightCurveFile(path[, quality_bitmask]) Defines a class for a given light curve FITS file from NASA’s Kepler and K2 missions.
TessLightCurveFile(path[, quality_bitmask]) Defines a class for a given light curve FITS file from NASA’s TESS mission.
